## Manuel eklentiler

Firebase bağlatı bilgilerinin tutulduğu config.xml dosyasının içeriği şu şekilde olmalıdır :

'''
<?xml version="1.0" encoding="utf-8" ?>

<FireBase>
	<ApiKey>ApiKey_Bilgisi</ApiKey>
	<AuthDomain>AuthDomain_Bilgisi</AuthDomain>
</FireBase>

'''
